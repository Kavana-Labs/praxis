# Praxis Style Guide

## Design Philosophy

- Academic
- Calm
- Precise
- Minimal

No visual noise. No gimmicks.

---

## Color Palette

- Primary: Academic Violet / Indigo
- Accent: Soft Lavender
- Neutral: Slate Gray, Off-White

Avoid neon or aggressive gradients.

---

## Typography

- Primary: Inter / IBM Plex Sans
- Code: JetBrains Mono / Fira Code
- Math: Computer Modern / KaTeX default

---

## UI Principles

- Content first
- Clear hierarchy
- Subtle shadows
- Consistent spacing
- Keyboard-first interactions

---

## Accessibility

- WCAG AA contrast minimum
- Keyboard navigation
- Screen reader-friendly math
- Scalable fonts

---

## Branding Usage

- Praxis logo should never overpower content
- Kavana Labs branding appears subtly in footers and legal sections
